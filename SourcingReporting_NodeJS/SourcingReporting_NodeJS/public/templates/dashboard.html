Release Infos

Release 1.4 - 18.06.2018

Neues in Version 1.4:

  • Persönliches Dashboard als Startseite
    Zahlen aus der Statistik-Seite herausgelöst und mit der für den aktuellen Monat aktuellen RememberMe-Liste ergänzt.
    Du kannst die persönliche Startseite mit dem epunkt-Logo erreichen.
  • Neue Statistik: Response Value - gibt an wie viele Responses wird erhalten haben und wie viele davon positiv und negativ waren
  • Neue Statistiken: Time-to-Piecharts wurden hinzugefügt
  • Bugfix: Kandidaten löschen ohne Admin-Rechte nun möglich
  • Bugfix: Response von Ja auf Nein stellen - Response-Value wird automatisch zurückgesetzt
  • Bugfix: Besetzungen je Team - Filter-Funktion repariert
  • Clean Code: Der Sourcecode wurde zusammengeräumt und sinnvollere Namen für Variablen und Funktionen vergeben (wodurch das halbe Tool zeitweise nicht mehr ging), aber zwegs Übersichtlichkeit und
    "Viki hat nix mehr gefunden"-Situation notwendig war.
  • Unsere Sourcing-Helden stellen sich vor...
  • Easter Egg versteckt

System-Info

Die Webanwendung wurde serverseitig mit NodeJS inkl express-Framework umgesetzt.
Clientseitig besteht das Service aus HTML5-Seiten, CSS und dem AngularJS-Framework.
Außerdem wurden für das Design Teile des Bootstrap-Frameworks verwendet.

Für die Statistiken wurde das Chart.JS Framework eingebunden.
