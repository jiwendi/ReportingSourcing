Release Infos

Release 1.5 - 20.07.2018

Neues in Version 1.5:

  • Kandidatensuche in der Nav-Bar rechts oben hinzugefügt
    • Suche nach Vorname ODER Nachname
    • Suche nach eR-Nummer
    • Suche nach Quelle-Zusatz (Link)
  • Alertify als Framework für die Alerts (Benachrichtigungen) eingefügt
    • Benachrichtigungen werden nun rechts unten übereinander gestaffelt angezeigt, statt rechts oben (übereinander)
    • Benachrichtigungen werden nun nach einer gewissen Zeit automatisch ausgeblendet (und bleiben nicht ewig stehen)
  • Einstellungen wurden ins Menü übernommen
    • Menü wird dadurch übersichtlicher
    • Teams, Quellen, Benutzer in Einstellungen verschoben
    • Kandidaten anonymisieren (DSGVO) in Einstellungen hinzugefügt - jedoch noch ohne Backend-Funktionalität
  • Responsive Design begonnen - Darstellung auf hp epunkt Laptops angepasst
  • Time-To-Max Listen hinzugefügt (für Fehlersuche bei Time-To (6M) Anzeige)
  • BUGFIX: RememberMe-Liste kann nun nach Datum sortiert werden
  • BUGFIX: Besetzungs-Liste kann nun nach Datum sortiert werden
  • myDashboard
    • myRememberMe-Liste: Anzeige aller Kandidaten bis zum aktuellen Monat, bisher wurde nur das aktuelle Monat angezeigt; nun werden auch die "älteren" RememberMe angezeigt
    • Übersicht meiner Telefonnotizen dieser Woche
    • Anzeige von Ansprache/nicht angesprochen (meine Woche)
    • Anzeige von Response (inkl. persönlicher Response Rate), Aufteilung auf positive und negative Response (inkl. %-Angabe)
  • Sourcing-Heros Übersicht hinzugefügt
  • Enter bei Login funktioniert nun

System-Info

Die Webanwendung wurde serverseitig mit NodeJS inkl express-Framework umgesetzt. Clientseitig besteht der Web-Service aus HTML5-Seiten, CSS und dem Angular-Framework. Die Seiten werden asynchron mit AJAX bei Bedarf nachgeladen.
Außerdem wurde für das Design das Bootstrap-Framework verwendet.

verwendete JavaScript-Frameworks:

  • Grundfunktionen:
    • Angular
    • jQuery
  • Statistiken:
    • Chart.JS
  • Alerts:
    • AlertifyJS
